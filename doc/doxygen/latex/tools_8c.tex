Element to processor map (non moving window)

Read in element to processor map information for non-moving window case. In this case, we use the mapping provided by genmap without modification.

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em out\_\-map}]Element mapping. Position-based. I.e. out\_\-map\mbox{[}0\mbox{]}=5 means element 0 is assigned to rank 5 \item[{\em nelgt}]number of global elements \item[{\em dim}]number of vertices per element \end{description}
\end{Desc}
